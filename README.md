PHP User Signup/Login System

A simple and professional user signup/login system built with PHP and MySQL. Includes password hashing and a clean blue UI.

 Features
 User registration
Secure password hashing
Login system
 Dashboard after login


How to Run
1. Copy the project folder to your XAMPP `htdocs` directory.
2. Make sure your database and `users` table exist (already set up if your app is working).
3. Update `db_connect.php` with your database credentials if needed.
4. Open in browser: `http://localhost/your-folder/signup.php`

 How to Use
1. Sign up a new account.
2. After registration, click **Login here**.
3. Login with your credentials to access the dashboard.

 Tech Stack
PHP 
MySQL
HTML/CSS (Blue theme)
 Password hashing for security
